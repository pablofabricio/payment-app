PaymentApp

Welcome to the front-end module of our payment management system. This module is designed to provide a seamless user interface for managing payments using credit cards within our comprehensive system.

- Installation
```sh
npm install 
```
- Run App
```sh
npm start
```

Run tests
```sh
npm run test
```
